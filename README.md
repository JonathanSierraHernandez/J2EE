# J2EE

The name of the project is Photo&Film4You and it is based on Java J2EE. This project is used in order to manage users and all related with his photos and videos (products, brands, rents, cancelations, etc).

Frontend is developed using Thymeleaf, JSF and HTML.

Backend is developed using EJB.

Integration Layer is developed using JPA, JDBC, PostgreSQL.
